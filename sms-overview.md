# SMS Overview

Displays the SMS sent to customers according to the set filters.

**Filters**

* SMS Type
  * Booking SMS
  * Customer SMS
  * Dynamic SMS
  * Destination App SMS
  * Select Offer SMS
  * Flight Time Change
* Sent period
* Phone number
* SMS Status
  * Queued
  * Sending
  * Sent
  * Failed
  * Received
