# Your Learning Path by Role

### Overview

Not everyone needs to read the entire manual. This page gives you a **recommended reading order** based on your role so you can focus on what's most relevant to you.

***

### 🧑‍💼 Booking Agent

Your main job is creating and managing bookings and communicating with customers.

**Start here:**

1. [Dashboard](https://manual.tourpaq.com/dashboard/dashboard) — understand your daily overview
2. [New Booking](https://manual.tourpaq.com/booking/new-booking) — create bookings step by step
3. [All Bookings](https://manual.tourpaq.com/booking/all-bookings) — search and manage existing bookings
4. [Customers](https://manual.tourpaq.com/customer/customers) — manage customer profiles
5. [Offers](https://manual.tourpaq.com/offers) — create and track offers
6. [Extras](https://manual.tourpaq.com/extras-setup/extras) — understand available add-ons
7. [E-tickets Overview](https://manual.tourpaq.com/e-tickets-overview) — send and track tickets

***

### ⚙️ System Administrator

Your focus is configuring the system, managing users, and maintaining integrations.

**Start here:**

1. [Users](https://manual.tourpaq.com/users/users) — set up and manage user accounts and roles
2. [Brands](https://manual.tourpaq.com/brands) — configure brands and their settings
3. [Setup → System Setup](https://manual.tourpaq.com/setup/system-setup) — core system configuration
4. [Setup → Destinations & Resorts](https://manual.tourpaq.com/setup/destination) — geographic configuration
5. [Security measures](https://manual.tourpaq.com/security-measures) — understand security settings
6. [2 Factor Authentication](https://manual.tourpaq.com/2-factor-authentication) — enforce secure logins
7. [GDPR](https://manual.tourpaq.com/gdpr-general-data-protection-regulation) — data protection compliance
8. [Restful API authentication](https://manual.tourpaq.com/restful-api-authentication) — API and integrations

***

### 💰 Finance / Accounting

Your focus is payments, reconciliation, invoicing, and profitability tracking.

**Start here:**

1. [Dashboard](https://manual.tourpaq.com/dashboard/dashboard) — financial summary widgets
2. [Payment Registration](https://manual.tourpaq.com/finance/payment-registration) — register incoming payments
3. [Payment File Import](https://manual.tourpaq.com/finance/payment-file-import) — bulk import payments
4. [Refund File Import](https://manual.tourpaq.com/finance/refund-file-import) — process refunds
5. [Invoice](https://manual.tourpaq.com/invoice) — manage invoicing
6. [Autobilling](https://manual.tourpaq.com/autobilling) — automated billing setup
7. [Missed Payments](https://manual.tourpaq.com/missed-payments) — identify and chase overdue payments
8. [Export](https://manual.tourpaq.com/export-1/export) — export financial reports

***

### 🏨 Hotel Manager / Supplier

Your focus is hotel setup, contracts, availability, and room management.

**Start here:**

1. [Hotels](https://manual.tourpaq.com/hotel/hotels) — overview of hotel management
2. [Hotel Creation](https://manual.tourpaq.com/hotel/hotel-creation) — set up a new hotel
3. [Hotel Contracts](https://manual.tourpaq.com/hotel-contracts) — manage supplier contracts
4. [Base Room Types](https://manual.tourpaq.com/base-room-types) — define room categories
5. [Stop Sales](https://manual.tourpaq.com/stop-sales) — close availability for specific dates
6. [Close Out](https://manual.tourpaq.com/close-out) — block out entire periods
7. [Special Offers](https://manual.tourpaq.com/special-offers) — configure hotel-level promotions
