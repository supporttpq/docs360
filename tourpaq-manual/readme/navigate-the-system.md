# Navigate the System

### Overview

This page gives you a guided tour of the Tourpaq interface so you can find what you need quickly.

***

### The Main Sidebar

The left sidebar is your primary navigation. It's organised into sections, each covering a major area of the system:

| Section              | What's in it                                           |
| -------------------- | ------------------------------------------------------ |
| 📊 **Dashboard**     | Real-time snapshot of sales, bookings, and performance |
| 🔔 **Notifications** | System alerts and reminders                            |
| 📋 **Booking**       | Create and manage customer bookings                    |
| 👤 **Customer**      | Customer profiles and history                          |
| 🎫 **Tickets**       | Print and manage e-tickets                             |
| 💰 **Finance**       | Payments, refunds, invoices, and exports               |
| 🏷️ **Price List**   | Pricing configuration and rules                        |
| 🏨 **Hotel**         | Hotel setup, contracts, and availability               |
| ✈️ **Transport**     | Flight and transport configuration                     |
| ✅ **Quality**        | Service cases and quality management                   |
| 📤 **Export**        | Data export lists and reports                          |
| 👥 **Users**         | User management, brands, roles, suppliers              |
| 🌍 **Guest App**     | Mobile destination guide content                       |
| ⚙️ **Extras Setup**  | Extras, discounts, insurance, and campaigns            |
| 📧 **Email Setup**   | Email and SMS templates and automation                 |
| 🔧 **Setup**         | System configuration, gateways, destinations           |

***

### The Top Bar

The top bar contains:

* 🔍 **Search** — Global search across bookings, customers, and more
* 🌐  **Brand selecto**r — Dropdown where a brand is selected
* 👤 **User menu** — Profile settings, 2FA, logout
