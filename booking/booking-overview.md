# Booking overview

### Overview

The **Booking** module is where you **create**, **find**, and **maintain** customer bookings in **Tourpaq Office**.

Use it for day-to-day tasks like creating new bookings, editing passengers, handling payments, sending tickets, and managing special cases like waitlists.

### Where to start

#### Create a new booking

Use [New Booking](new-booking/) when you need to build a booking from scratch.

Typical flow:

1. Search transport + hotel availability.
2. Take allotment.
3. Add passengers and extras.
4. Review price, cost, and payments.
5. Save and send confirmations.

For the step-by-step screen walkthrough, see [New Booking (screen)](new-booking/new-booking/).

#### Find and edit existing bookings

Use [All bookings](all-bookings/) to search, filter, and open booking details.

Use it when you need to:

* Find a booking by date, customer, hotel, transport, owner, or status.
* Export or report on bookings using Statistics and Totals.
* Open a booking and continue work in tabs like Economics, Passenger details, Comments, and History.

### Common booking actions

#### Payments and financial overview

Use [Economics](new-booking/economics.md) to check deposits, instalments, due dates, payment status, and refund eligibility.

#### Copy or reschedule a booking

* Use [Copy Booking](new-booking/copy-booking.md) to reuse the passenger list on a new booking draft.
* Use [Moved Booking](new-booking/moved-booking.md) to link old and new booking numbers when rescheduling.

#### Handle sold-out departures

Use [Waiting list (WL)](new-booking/waitting-list.md) to create waitlist bookings when seats/beds are sold out.

#### Remove a passenger from one direction only

Use [Remove pax on outbound or homebound only](new-booking/remove-pax-on-outbound-or-homebound-only/) when a passenger is a no-show for outbound or homebound.

### FAQ

#### Where do I send tickets or confirmations?

Use the booking’s **E-mails** tab for booking emails. Use the Tickets area for printing and sending ticket PDFs.

#### What’s the difference between **All bookings** and **New Booking**?

* **New Booking** creates a new booking draft.
* **All bookings** finds existing bookings and opens booking details for edits and reporting.

#### I can’t find availability in New Booking. What should I check?

Most issues are missing setup:

* wrong **Brand**
* missing **allotment** (transport or hotel)
* missing **price list** for the period

Start with widening filters and confirming Brand selection.
